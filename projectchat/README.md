# Bank Enquiry Chatbot

A complete web-based banking chatbot with bilingual support (English/Tamil), voice input, and text-to-speech features.

## Features

✅ User Authentication (Login/Register)
✅ Rule-based Chatbot with FAQ knowledge base
✅ Bilingual Support (English & Tamil)
✅ Voice Input (Speech-to-Text)
✅ Text-to-Speech for responses
✅ Modern WhatsApp-style UI
✅ Chat history saving
✅ Responsive design
