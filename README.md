# 🏦 Bank Enquiry Chatbot with Voice AI

A smart banking chatbot built using **Python (Flask)** that can answer user queries related to banking services.  
This chatbot supports **Tamil and English languages**, along with **voice input and voice output** features.

---

## 🚀 Features

- 💬 Chat-based banking assistance
- 🌐 Multilingual support (Tamil + English)
- 🎤 Voice Input (Speech Recognition)
- 🔊 Voice Output (Text-to-Speech using gTTS)
- 🧠 Smart keyword-based response system
- 💾 Chat history storage
- 🔐 User Authentication (Login & Register)

---

## 🛠️ Technologies Used

- Python (Flask)
- HTML, CSS, JavaScript
- SQLite Database
- gTTS (Google Text-to-Speech)
- Web Speech API (Speech Recognition)
